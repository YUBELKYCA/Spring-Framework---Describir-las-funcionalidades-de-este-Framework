Spring Framework Functionalities

1. Inversion of Control (IoC)

IoC promotes loose coupling by using dependency injection.

        <bean id="myService" class="com.example.MyService">
            <constructor-arg ref="myDependency"/>
        </bean>

        <bean id="myController" class="com.example.MyController">
            <property name="myService" ref="myService"/>
        </bean>

2. Aspect-Oriented Programming (AOP)

AOP enables modular cross-cutting concerns like logging and security.

        <aop:aspectj-autoproxy>
            <aop:include pointcut="within(com.example.MyPackage..*)"/>
        </aop:aspectj-autoproxy>

        <aop:advisor ref="myExceptionHandler">
            <aop:pointcut expression="execution(* *(..))"/>
        </aop:advisor>

3. Exception Handling

Centralized exception handling simplifies error management.

        <bean id="myExceptionHandler" class="com.example.MyExceptionHandler">
            <property name="target" ref="myService"/>
        </bean>
